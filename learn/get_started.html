This example does a really quick introduction on getting something up and running. It does not explain all the details, those will be covered in another tutorial.

Ensure that you have followed installation instructions. If everything is setup correctly, then running:

	go run gioui.org/example/hello@latest

Should display a pretty "Hello, Gio!" message.

Creating a new package

If you are unfamiliar with Go, then more help can be found at go.dev/learn.

First step in creating a Go program requires setting up the module.

We'll use gio.test as our module name, however, it's recommended to use a repository name when you want to upload it. The module name can be later changed.

	go mod init gio.test

Creating the program

Let’s create main.go with the following code:

	package main

	import (
		"log"
		"os"

		ui "github.com/utopiagio/utopia"
	)

	var win *ui.GoWindowObj

	func main() {
		app := ui.GoApplication("ColorPicker")
		win = ui.GoWindow("ColorPicker Demo")
		win.SetLayoutStyle(ui.VFlexBoxLayout)
		win.SetMargin(10,10,10,10)
		win.SetBorder(ui.BorderSingleLine, 2, 10, ui.Color_Blue)
		win.SetPadding(10,10,10,10)
		
		// create a new layout to display window content
		contentView := ui.GoVFlexBoxLayout(win.Layout())
		// allow the layout to expand to fill the parent window layout 
		contentView.SetSizePolicy(ui.ExpandingWidth, ui.ExpandingHeight)
		contentView.SetMargin(5,5,5,5)
		contentView.SetBorder(ui.BorderSingleLine, 1, 10, ui.Color_Blue)
		contentView.SetPadding(5,5,5,5)
		
		// create a new layout at the bottom of the window
		bottomView := ui.GoHFlexBoxLayout(win.Layout())
		// allow the layout to expand to fill the width of the parent window layout but maintain a fixed height
		bottomView.SetSizePolicy(ui.ExpandingWidth, ui.FixedHeight)
		bottomView.SetSpacing(ui.SpaceStart)
		bottomView.SetMargin(5,5,5,5)
		bottomView.SetBorder(ui.BorderSingleLine, 1, 10, ui.Color_Blue)
		bottomView.SetPadding(5,5,5,5)
		
		
		// add a button to the bottom bar
		btnClose := ui.GoButton(bottomBar, "Close")
		btnClose.SetMargin(5,0,0,0)
		btnClose.SetBorder(ui.BorderSingleLine, 1, 5, ui.Color_Blue)
		btnClose.SetPadding(4,4,4,4)
		btnClose.SetOnClick(ActionExit_Clicked)
		btnClose.SetSizePolicy(ui.FixedWidth, ui.FixedHeight)
		
		
		win.Show()
		app.Run()
	}

	func ActionExit_Clicked() {
		log.Println("ActionExit_Clicked().......")
		os.Exit(0)
		//win.Close()
	}

Let’s then update all the dependencies with:

	go mod tidy

Once that succeeds, the program should start up with:

	go run .

Creating the window

Every program requires an application object, the main function creates the application instance and then creates the main window. It then displays the main window and runs the application loop that talks to the operating system

	func main() {
		// create application instance before any other objects
		app := ui.GoApplication("ColorPicker")
		// create application window
		win = ui.GoWindow("ColorPicker Demo")
		// set a layout style for the window (Vertically stacked widgets)
		win.SetLayoutStyle(ui.VFlexBoxLayout)
		// add an indentation for the layout view within the window
		win.SetMargin(10,10,10,10)
		// add a border to the layout view
		win.SetBorder(ui.BorderSingleLine, 2, 10, ui.Color_Blue)
		// add padding to the layout view
		win.SetPadding(10,10,10,10)

		// display the application window
